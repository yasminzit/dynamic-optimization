# dynamic-optimization
Applied Dynamic Optimization &amp; Decision Modeling

Практика решения прикладных задач динамической оптимизации и оптимального управления в экономике

Условия задач - файл doef_practice_2025.pdf

### Задачи
- Оптимальное управление с ограничениями (bang-bang)
- Максимизация NPV инвестиционных проектов
- Минимизация времени достижения целевого состояния
- Сценарный анализ параметров моделей

### Инструменты
Python (numpy, scipy), Excel (Solver), визуализация
